Laboratorio #5 — IE-0117 Programación Bajo Plataformas Abiertas
Descripción del contenido
Este repositorio implementa cuatro estructuras de datos clasicas en lenguaje C: Arreglo Dinamico, Lista Enlazada Simple, Lista Doblemente Enlazada y Stack. Cada una esta en su propio .c/.h, y main.c demuestra todas las operaciones.

Pasos para compilar
    make

Esto genera el ejecutable main. Para limpiar:

    make clean

Pasos para ejecutar
    ./main
